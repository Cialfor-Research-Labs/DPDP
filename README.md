# DPDPA Compliance Engine

Skeletal monorepo scaffold for a DPDPA compliance platform based on the provided architecture:

- `frontend/`: React + TypeScript dashboard scaffold
- `backend/`: FastAPI service scaffold

This repo is intentionally light on implementation and heavy on boundaries so each component can evolve independently.

Backend dependencies are managed with `requirements.txt` for now, and the project is scaffolded for local development without Docker.
